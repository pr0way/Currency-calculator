# Currency Calculator

Based on [NBP API](http://api.nbp.pl/).
Supported conversion - at the moment only: **PL** -> any other.

## Project setup

```
yarn install
```

### Development

If you want to be a contributor, clone this repo and in root directory run:

```
yarn run serve
```
